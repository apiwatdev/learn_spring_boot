# Spring Boot Learning Repository

This repository is a collection of projects and tutorials for learning Spring Boot.

## Description

This repository contains a series of projects and tutorials for learning Spring Boot. It covers several topics, including building REST services, using Spring Security and Angular, working with OAuth2, using Kotlin Coroutines and RSocket, and building web applications with Spring Boot and Kotlin.

## Projects

### Building REST Services

This project demonstrates how to build REST services using Spring Boot. It includes examples of how to create REST endpoints, handle exceptions, and use Spring Data JPA to interact with a database.

### Spring Security and Angular

This project demonstrates how to use Spring Security with Angular. It includes examples of how to create a login page, protect resources, and use JWT tokens for authentication.

### Spring Boot and OAuth2

This project demonstrates how to use OAuth2 with Spring Boot. It includes examples of how to configure OAuth2 clients and servers, use JWT tokens, and integrate with Spring Security.

### Spring Boot with Kotlin Coroutines and RSocket

This project demonstrates how to use Kotlin Coroutines and RSocket with Spring Boot. It includes examples of how to create a reactive service layer, use RSocket for communication, and handle errors.

### Building Web Applications with Spring Boot and Kotlin

This project demonstrates how to build web applications with Spring Boot and Kotlin. It includes examples of how to create a web application, handle HTTP requests, and use Thymeleaf for templating.
